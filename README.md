# 餐廳清單
簡單建立8間餐廳的詳細資訊
# 功能描述
- 點擊餐廳可顯示餐廳的詳細資訊
- 可依照名稱搜尋餐廳
- 點擊navbar左側"我的餐廳清單"，可回到首頁
# 環境建置與需求
- Node.js: v10.15.0
- Express: v4.17.1
- Express-Handlebars: V5.1.0
# 安裝與執行步驟
1. 下載專案
> git clone https://github.com/ShengYaoHuang/restaurant_list.git

2. 進入專案資料夾
> cd restaurant_list

3.下載套件
> npm install

4. 執行程式
> npm run dev
